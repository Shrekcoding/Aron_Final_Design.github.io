About Zeldman

What is HTML?

Click the image to learn more about HTML!

HTML stands for Hyper Text Markup Language and is the standard markup language for creating web pages.
Html is used to describe the structure of the page and consists of many elements and variables that tell the browser how to display the context.
Html labels peices of content that are easy to follow, such as "p" for paragraph, or "h1" and "h5" for its largest and smallest headings.

What is CSS?

Click the image to learn more about CSS!

CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files.
